DellDi API

使用自然语言查询您的数据库

API 文档

DellDi API 提供了一套强大的接口，用于将自然语言转换为 SQL 查询并执行。

Swagger UI 文档 ReDoc 文档

主要功能

  • 自然语言转 SQL
  • 执行 SQL 查询
  • 数据可视化
  • 训练与优化
  • 问题推荐

快速开始

  1. 访问 /docs 查看完整 API 文档
  2. 使用 /api/v0/generate_questions 获取推荐问题
  3. 使用 /api/v0/generate_sql 将问题转换为 SQL
  4. 使用 /api/v0/run_sql 执行 SQL 查询
